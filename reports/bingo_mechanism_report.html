一、项目来源与归因声明（新增，首要章节）

核心事实：
  • BinGO 的底座是清华大学开源项目 OpenMAIC（Open Multi-Agent Interactive Classroom，仓库 github.com/THU-MAIC/Bingo，附 JCST 2026 论文）。
  • Git 远程：origin = THU-MAIC/OpenMAIC.git（上游）；bingo = caijiahangvip-sudo/BinGO.git（参赛者 fork）。
  • BinGO 不是从零发明，而是基于 OpenMAIC 的二次开发工程。

BinGO 的贡献定位：

BinGO 把 OpenMAIC 工程化为 Windows 一键安装、本地/离线可运行的桌面产品，并补充 AMD ROCm 本地推理、Windows 凭据管理器密钥存储、系统代理适配、自动更新与中文本地化能力。

三、OpenMAIC 上游提供的教学机制（归因纠正）

归因纠正声明：本节内容在 v1.0 报告中被描述为"BinGo 机制"，这是错误的。以下机制均来自 OpenMAIC 上游，BinGO 未自研这些教学机制，仅在其基础上完成部署、本地化和使用流程扩展。

3.1 多智能体课堂编排（OpenMAIC 上游）

上游 ICAP 辩论、Teach-back 视觉推进、请求澄清等课堂编排机制，由上游 lib/orchestration/director-graph.ts 实现（含 discussionMode: 'debate'、request_clarification、wait_for_user_teaching）。BinGO 未修改其核心编排逻辑。

3.2 动作引擎与安全校验（OpenMAIC 上游）

上游 动作越界拦截、安全动作集、有效动作计算，由上游 lib/action/engine.ts 实现（validateAction、SAFETY_ACTIONS、getEffectiveActions）。BinGO 未自研动作安全引擎。

3.3 学习画像引擎（OpenMAIC 上游）

上游 接入 ICAP 证据、跨课时画像生成，由上游 lib/learning/profile-engine.ts 实现。BinGO 未自研学习画像。

3.4 课堂生成、回放、白板账本、课件导出（OpenMAIC 上游）

上游 大纲/场景/幻灯片生成（lib/generation/）、结构化回放（lib/playback/）、白板账本（lib/store/canvas.ts 的 WhiteboardActionRecord）、PPTX 导出（lib/export/，用第三方 pptxgenjs）均来自上游。

3.5 PDF/语音/向量集成（OpenMAIC 上游集成第三方）

上游集成 第三方 上游集成了 MinerU（PDF）、SenseVoice（ASR）、CosyVoice（TTS）、向量检索。BinGO 仅补充了本地服务启动脚本。

五、BinGO 自行新增或深度改造的工程能力

以下每项均给出代码路径，为 BinGO 真实且可验证的贡献。

5.1 Windows 桌面运行时（Tauri+Rust）

BinGO 文件：src-tauri/src/lib.rs（608 行）、src-tauri/tauri.conf.json。把 Web 端打包为 Windows 桌面应用，捆绑 node.exe+server.js。提交：5a9e9d3。

5.2 Windows 凭据管理器存储密钥

BinGO 文件：src-tauri/src/lib.rs。desktop_secret_write(L435, CredWriteW)、desktop_secret_read(L361)、desktop_secret_delete(L471)。API Key 存入 OS 凭据库（CRED_PERSIST_LOCAL_MACHINE），非明文。测试：validates_secret_targets。

5.3 WinHTTP 系统代理与 PAC 适配

BinGO 文件：src-tauri/src/lib.rs。windows_system_proxy(L220)、normalize_proxy_address(L182)。用 WinHttpGetIEProxyConfigForCurrentUser、WinHttpGetProxyForUrl（PAC 自动配置）。测试：normalizes_windows_proxy_formats（4 种格式）。

5.4 Job Object 后台进程回收

BinGO 文件：src-tauri/src/lib.rs。attach_kill_on_close_job(L34) 用 CreateJobObjectW+JOB_OBJECT_LIMIT_KILL_ON_JOB_CLOSE，应用退出自动回收子进程。

5.5 多服务捆绑启动 + 5 次重试 + 健康检查

BinGO 文件：src-tauri/src/lib.rs。start_bingo_server(L497, 5 次重试)、wait_until_ready(L134, 24s 超时)、health_is_ready(L110, 校验 status/desktop/version)、会话令牌 desktop_session_token。提交：8ee751c。

5.6 自动更新与 NSIS 安装器

BinGO 文件：src-tauri/tauri.conf.json、.github/workflows/release-desktop.yml、DESKTOP_RELEASE.md。Tauri updater 签名校验 + GitHub Actions 自动构建。

5.7 WSL + AMD ROCm 本地模型启动器

BinGO 文件：scripts/start-bingo.ps1（377 行）、app/api/local-services/configure-rocm/route.ts。编排 4 个本地服务（CosyVoice 50000/SenseVoice 50001/MinerU 50002/Embedding 50003），WSL 端口探测。提交：d35c78c。让无 N 卡用户经 ROCm 跑本地模型。

5.8 中文本地化能力集

BinGO 新华字典检索（app/api/chinese-xinhua/，已验证 200）、教材下载器（app/api/textbooks/*）、本地备份（app/api/local-backup，已验证 200）、桌面运行时门（components/desktop-*）。
